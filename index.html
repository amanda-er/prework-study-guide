CSS

  • Margin : indicates how much space we want around the outside of an element
  • Padding : indicates how muuch space we want around the content inside an element
  • CSS : language used by developers to define how webpage content is presented to the users
  • CSS is used to define style rules for specific elements of an HTML page and creates a format with different style options
  • Inline CSS : applies to specific HTML tag and each one must be eddited individually whenever you make a design change
  • Internal Style Sheet : contains CSS rules for the webpage in the head section of the HTML file
  • External CSS Style Sheet : allows developers to keep all their CSS rules in a separate file thus making design changes easier
  • Declarations : contain 2 important components, CSS property we wnat to apply and value for that property
